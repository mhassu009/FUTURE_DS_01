# Business Sales Performance Analytics
**Data Science Internship - Task 01 (Future Interns)**

## Project Overview
This project focuses on analyzing business sales data to identify revenue trends, top-selling products, and regional performance across different cities like Karachi, Lahore, and Islamabad.

## Key Insights
* **Top Product:** iPhone (Highest revenue contributor)
* **Leading Category:** Electronics
* **Total Revenue:** 1,679,000

## Tools Used
* **Microsoft Excel:** Data cleaning, Pivot Tables, and Data Visualization.
